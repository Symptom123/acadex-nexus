# ACADEX Smart School Nexus

ACADEX is an innovative, AI-powered educational management platform that reimagines how schools operate. It provides a unified experience for administrators, teachers, students, and parents.

## Key Features

- **Role-Based Portals**: Tailored dashboards for Administrators, Teachers, Students, and Parents.
- **Smart Predictions & Analytics**: Identify trends, track performances, and support student success using intelligent analytics.
- **Academic Management**: Seamlessly handle class attendance, assignments, grades, and report card generation.
- **Next-Gen UI**: Designed with a dynamic, hacker-chic aesthetic featuring responsive glassmorphism, fluid animations, and custom styling.

## Technologies Used

- React (Vite)
- TypeScript
- Tailwind CSS (with custom utility tokens and advanced animations)
- Framer Motion for animations
- Recharts for data visualization
- shadcn/ui component library

## Getting Started

To run the project locally:

```sh
# Clone the repository
git clone https://github.com/Symptom123/acadex-nexus.git

# Navigate to the project directory
cd acadex-nexus

# Install the dependencies
npm install

# Start the development server
npm run dev
```
